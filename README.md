# nextjs-deep-dive

This folder is study material for Next.js 16—mostly markdown notes, exercises, and templates. There isn’t a single app you `npm install` here; it’s reference you use while building something real in another project.

**Main reference:** [`nextjs-study-guide.md`](nextjs-study-guide.md) — cheat-sheet style notes on the App Router, Server vs client components, routing, data fetching, ISR, auth/middleware, security and performance checklists, deployment gotchas, and a bunch of copy-paste snippets plus interview Q&A.

**Broader map:** [`master-index.md`](master-index.md) lists the other files (e.g. SaaS architecture template, exercises) and suggested order if you want the full path.

Official docs are still the source of truth: [Next.js](https://nextjs.org/docs), [React](https://react.dev).
